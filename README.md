# Assessing-ML-Models-in-Landslide-Susceptibility-and-Risk-Mapping-in-Jammu-Region-India

Resources include:
	(1) Table for Analytic Hierarchy Process (AHP) for Landslide Susceptibility Zonation,
	(2) Data sheet for computing the Area Under Curve (AUC) for different ROCs for AHP, Log. R, ANN & RF models,
	(3) R Code for plotting combined ROCs based on (2), and
	(4) R Notebook for RF-based LSI Classification - including model assessment and global SHAP analysis.

Paper to be presented at the International Geoscience and Remote Sensing Symposium (IGARSS), 2026, in Washington, D.C., USA (9-14 Aug 2026).
